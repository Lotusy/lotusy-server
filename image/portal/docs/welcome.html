Lotusy Image RESTFul Services
HOST - <https://{env}.image.lotusy.com/rest/>


Image API contains 2 main categories:
  • User
  • Business
  • Comment

User APIs - handles all user image related request
  1. display a user current profile image
  2. display a user profile image
  3. get all profile image links of a user
  4. get comment image links of a user
  5. upload a user profile image
Business APIs - handles all business image related request
  1. display a business profile image
  2. upload a business profile image
  3. get comment image links of a business
Comment APIs - handles all rating related request
  1. display an image of a comment
  2. upload an image to a comment
  3. get image links of a comment


For more details about each API, please click on the API link on the left.

API detail contains:
  • What the API does. (e.g. Process a Credit Card Online Payment)
  • API method. (e.g. POST)
  • API uri. (e.g. /web/payment/{user_id}/creditcard)
  • Header information (e.g. "Content-Type: application/json")
  • Request body information (e.g. {"save_card":true,"order":{"order_id":"13fasiudbvahserhawer"}})
  • Request response success and failed messages
