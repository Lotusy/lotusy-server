Lotusy Comment RESTFul Services
HOST - <https://{env}.comment.lotusy.com/rest/>


Comment API contains 2 main categories:
  • Comment
  • Reply

Comment APIs - handles all comment related request
  1. create a comment
  2. get a comment information
  3. delete a comment
  4. like a comment
  5. dislike a comment
  6. get comments near a location
  7. get comments by a user
  8. get comments of a business
Reply APIs - handles all reply related request
  1. create a reply
  2. get reply count of a comment
  3. get replies of a comment


For more details about each API, please click on the API link on the left.

API detail contains:
  • What the API does. (e.g. Process a Credit Card Online Payment)
  • API method. (e.g. POST)
  • API uri. (e.g. /web/payment/{user_id}/creditcard)
  • Header information (e.g. "Content-Type: application/json")
  • Request body information (e.g. {"save_card":true,"order":{"order_id":"13fasiudbvahserhawer"}})
  • Request response success and failed messages
