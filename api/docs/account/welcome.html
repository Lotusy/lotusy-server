Lotusy Accounts RESTFul Services
HOST - <https://{env}.account.foodster.club/rest/>


Payment API contains 2 main categories:
  • Token
  • User

Token APIs - handles all token related request
  1. register an account
  2. authorize an account
  3. get access token information
User APIs - handles all user related request
  1. get current user profile
  2. get a user profile
  3. update current user profile


For more details about each API, please click on the API link on the left.

API detail contains:
  • What the API does. (e.g. Process a Credit Card Online Payment)
  • API method. (e.g. POST)
  • API uri. (e.g. /web/payment/{user_id}/creditcard)
  • Header information (e.g. "Content-Type: application/json")
  • Request body information (e.g. {"save_card":true,"order":{"order_id":"13fasiudbvahserhawer"}})
  • Request response success and failed messages
