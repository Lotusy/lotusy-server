Lotusy Business RESTFul Services
HOST - <https://{env}.business.lotusy.com/rest/>


Business API contains 2 main categories:
  • Business
  • Rating

Business APIs - handles all business profile related request
  1. create a business
  2. get business profile
  3. get businesses near a location
Rating APIs - handles all rating related request
  1. get rating info of a business
  2. rate a business
  3. get the rating count of a business


For more details about each API, please click on the API link on the left.

API detail contains:
  • What the API does. (e.g. Process a Credit Card Online Payment)
  • API method. (e.g. POST)
  • API uri. (e.g. /web/payment/{user_id}/creditcard)
  • Header information (e.g. "Content-Type: application/json")
  • Request body information (e.g. {"save_card":true,"order":{"order_id":"13fasiudbvahserhawer"}})
  • Request response success and failed messages
